# OmniFlow

A mindmap-based todo/note management system where tasks ARE notes.

## Features

- 🧠 Mindmap visualization with multiple layout algorithms
- 📝 Rich markdown editing with CodeMirror
- 🔄 Local-first with Replicache (optional sync)
- 🏷️ Tags, references, and progress tracking
- ⌨️ Full keyboard shortcuts
- 📱 Mobile responsive

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Tech Stack

- React 19.2 + TypeScript
- Vite 7.2
- Tailwind CSS 4.1
- Replicache 15
- React Flow 11
- CodeMirror 6
